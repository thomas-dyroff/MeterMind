Design System · v1.0

Klare Zahlen, ruhige Flächen.

Das visuelle System für MeterMind — eine App, die Strom-, Gas- und Wasserverbrauch auf einen Blick verständlich macht. Jede Farbe, Form und Type-Entscheidung dient einem Ziel: dem Zählerstand Vorrang geben.
