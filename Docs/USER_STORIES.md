# USER STORIES
## Core
* Als Nutzer möchte ich einen Zähler anlegen, um meinen Verbrauch zu verfolgen.
* Als Nutzer möchte ich einen Zählerstand erfassen, um meinen aktuellen Verbrauch zu dokumentieren.
* Als Nutzer möchte ich meinen Verbrauch über Zeit sehen.
* Als Nutzer möchte ich meine Daten exportieren.

## Premium
* Als Nutzer möchte ich Kosten berechnen, um meine Ausgaben zu verstehen.
* Als Nutzer möchte ich Erinnerungen erhalten.
* Als Nutzer möchte ich meine Daten synchronisieren.
* Als Nutzer möchte ich Zähler per Foto erfassen.
